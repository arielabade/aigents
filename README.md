# AI Agents Portfolio

This repository is a portfolio-style collection of Python AI apps focused on modern SaaS chatbot and assistant workflows.

## Portfolio Structure

- `chat_between_ai/`
  - Debate simulator between OpenAI and Ollama personas.
- `image_generator/`
  - AI tourism poster generator.
- `summarizers/llmsAssistants/`
  - OpenAI and Ollama summarizers, technical assistants, and brochure generator.
- `gradio_user_interface/`
  - Historical notebooks and UI assets from previous iterations.
- `agrochemical_specialist/`
  - Domain-focused notebook assistant.
- `paid_media_specialist_chatbot/`
  - Domain-focused paid media chatbot notebook.

## Visual Direction

All refactored Python interfaces now follow a portfolio SaaS visual language:

- Pastel AI palettes (`AI Clean`, `AI Premium Pastel`, `AI Human`)
- High-contrast CTA buttons for conversion clarity
- Light, editorial UI composition with markdown-first outputs
